= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-improved-post-object` folder into your `wp-content/plugins` folder
2. Activate the `Advanced Custom Fields: Improved Post Object` plugin via the plugins admin page
3. Create a new field via ACF and select the Improved Post Object type
4. Please refer to the description for more info regarding the field type settings
